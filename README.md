# 🎵 Spotify Portfolio

A developer portfolio website inspired by Spotify's sleek interface. Browse projects like playlists, experience smooth animations, and enjoy a familiar yet unique way to showcase work.

![React](https://img.shields.io/badge/React-19-61DAFB?logo=react&logoColor=white)
![TypeScript](https://img.shields.io/badge/TypeScript-5.7-3178C6?logo=typescript&logoColor=white)
![Vite](https://img.shields.io/badge/Vite-6.1-646CFF?logo=vite&logoColor=white)

## ✨ Features

- 🎨 Spotify-inspired dark theme UI
- 🎵 Interactive "now playing" project showcase
- 💚 Like/unlike projects
- 📱 Fully responsive design
- ⚡ Fast and smooth animations

## 🚀 Quick Start

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build
```
